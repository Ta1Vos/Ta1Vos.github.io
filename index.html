    Five Nights of Judgement
    Five Nights of Judgement

    This is the 8th project, in period 6. This is the first project where I worked in a pair in the second year.

    • For this project we've experienced with the MVC (Model View Controller) model. This ensured we could advance in the way of programming server-sided, while also keeping a certain level of organization.
    • We also focused on OOP (Object Orientated Programming) with this project, which was used for the categories, products, reviews, etc.
    • We were required to work with Users on a deeper level for this project, including a website administrator. This user is able to edit most of the website's content so they are not reliant on a developer who would hard code content otherwise.

    On this project we had to design a review website, which includes categories (such as a brand) and products (such as phones). We were allowed to pick a subject on our own, so we went with Five Nights At Freddy's.

    I have challenged myself multiple times during this project, which includes:

    • A visit counter for every page. This would keep track of popular categories and products.
    • For the website administrator, a handy way of viewing the website as an User, Member and Admin.
    • I've added the possibility of uploading images for the website administrator. This wasn't required and I've learnt this feature after my preparations for the SkillsHeroes.
    • The website administrator is also capable of searching members & moderating, another feature I've added as an addition. The information gets loaded into a select input and they can be filtered on username.

    Unfortunately due to the amount of work I had on my hands and the not-so-actively-working team member, we've failed to finish all the features within the time limit. We have missed out on a few features which would've been nice to have, but didn't get to. Fortunately this didn't affect my grade at a huge amount.
